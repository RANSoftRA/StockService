* Services:
  (1) GET - All stocks
  (2) GET - Stock history (last month/week from now back)
  (3) POST - Register User (Hinzufügen)
  (4) PUT - Update User
  (5) GET - User Transactions
  (6) POST - Add Transaction - Buy or Sell
  
Calls:
(1) /finance/stocks
(2) /finance/stocks/{stocksymbol}/history
(3) /users?un=username&pw=password
(4) /secured/users/{username}&pw=pw
(5) /secured/users/{username}/transactions
(6) /secured/finance/transactions?stocksymbol=iwie&amount=20&isSell=true/false


** Alle Calls unter /secured müssen einen HTTP-Basic Authentication header mitschicken.
